# PokeCasino-V2
Proyecto Final TFG de Marcos Emilio Rodriguez y Alejandro Plata

# Descripción
Versión mejorada del proyecto de fin de curso.
Arreglaremos errores e implementaremos funciones nuevas tales como: Combate por turnos entre jugadores, Intercambios entre jugadores (agregando la mecánica de evolución por intercambio), creación de equipos y elegir los movimientos de los Pokémons (junto a agregar la posibilidad de comprar MTs y MOs en la tienda).
